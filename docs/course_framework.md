# 講義フレームワーク要約

このファイルは、講義資料PDFそのものをリポジトリに含めず、MVPが扱う分析フレームワークを要約して管理するためのものです。実際の提出時は、担当教員の配布資料・指示を優先してください。

## 因果のマトリクス

企業比較では、財務指標を単体で並べるだけでなく、原因、中間要因、財務への影響、確認データを対応づけて整理します。

- 需要環境: 利用頻度、客単価、稼働率が売上高成長率や営業利益率に影響する。
- 費用構造: 変動費率、固定費、設備や店舗の稼働が損益分岐点、安全余裕率、営業レバレッジに影響する。
- 資産効率: 設備、店舗、機材、在庫、資本構成がROA、ROE、総資産回転率に影響する。
- 財務安定性: 短期支払能力、自己資本、長期資金との対応が流動比率、自己資本比率、固定長期適合率に影響する。

## 企業分析9視点

MVPでは以下の9視点をWordレポートに出力します。

1. 市場
2. 顧客
3. 競合
4. 商品・サービス
5. 価格
6. チャネル
7. オペレーション
8. 財務
9. リスク

## 財務分析

収益性分析では、売上高成長率、営業利益率、ROA、ROE、総資産回転率、ROA/ROE分解を扱います。

財務安定性分析では、流動比率、自己資本比率、負債比率、固定比率、固定長期適合率を扱います。

＋α分析では、損益分岐点、安全余裕率、営業レバレッジ、4P/4C、顧客関係、バリューチェーン、FCF、PER/PBRを扱います。データ不足の場合は「推定不可」と明記します。

## 業種別の補足観点

カフェ・外食では、店舗網、立地、客単価、FC/直営比率、原材料費、人件費、リピート利用を重視します。

航空会社では、燃油費、為替、機材投資、路線網、搭乗率、国際線/国内線構成、外部環境の変化を重視します。

## 投資助言ではないこと

本MVPの出力は学習目的の企業比較レポートです。株式の売買判断や投資推奨を目的としません。
